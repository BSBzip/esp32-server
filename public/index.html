About Devices

Each ESP32 registers automatically by MAC address when it connects to WiFi. Online = checked in within the last 15 seconds. Select a specific device in the Commands tab to target it directly, or leave on "All Devices" to broadcast to every connected ESP32.

Deploy to ESP32

After uploading firmware above, press this button to send the update command to the device selected in the Commands tab (or all devices if none selected).

How to update the ESP32 wirelessly
  1. Make your code changes in VS Code
  2. Click the ✓ checkmark in VS Code to build
  3. Upload the firmware.bin file using the form above
  4. Press Send Update to ESP32 — no need to go to Commands
  5. The ESP32 downloads and installs automatically, then reboots
